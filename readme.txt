== Description ==

paysera.com payment plugin for wordpress.

paysera.com (Mokejimai.lt) is a leading payment provider in a Baltic region. Via this system you could accept payments via more than 20 different payment types. To use this plugin you need to register at paysera.com and create your project. You will get your project id, which should be written in this plugin settings.

== Installation ==

1. Install evp-event-v5.zip wordpress admin panel.
2. Activate the plugin. Find Evp: new in admin menu. Click it and fill all the necessary information

If you having any problems, write to support@webtopay.com
